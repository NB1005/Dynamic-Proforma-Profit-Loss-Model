# Dynamic-Proforma-Profit-Loss-Model
This simplified proforma Profit & Loss (P&L) and cash flow spreadsheet was created to help financial leadership understand, monitor, and communicate the expected cash impact of a major divestment project over time.

The model is designed to be *dynamic* and to follow a single, consistent scenario from start to finish across the full duration of the project. It provides a straightforward structure for projecting cash inflows and outflows by period, translating key assumptions into a clear view of timing, magnitude, and overall cash position. The company found it most useful when trying to compare divestment decisions from exiting regions to pruning portfolio product lines.

## Use Cases:
- What would happen if we exit "X" region?
- What would happen if we shut down "X" factory?
- What would happen if we EOL "X" Product/Model
- What would happen if we launch "X" product?
- and many more

In practice, teams found the model most useful when it was treated as an assumption-driven planning tool: where inputs were unknown, the spreadsheet is intended to capture reasonable estimates and make those assumptions explicit. At the same time, several sections are built to incorporate known, confirmed values as they become available, including realized divestment proceeds and actual investment or one-time cash costs. This combination supports both early-stage forecasting and ongoing updates as the project progresses and better information is obtained.
